## Decision

Every prompt instructs its agent to WRITE its output to the spec-defined path; the deliverable is the file on disk, not the reply. Source: _rules.md Conventions.

## Context

Foundational authoring convention (D3), baselined before the spine was built. Indexed in _decisions.md Decision index (D1–D4 → _rules.md Conventions).
