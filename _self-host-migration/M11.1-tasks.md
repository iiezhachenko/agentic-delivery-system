## SHARED CONTRACT A ‚Äî the caveman register (the rewrite target; reused by all M11 rewrite parts)

Rewrite ALL prose to this register. **Substance is invariant ‚Äî only fluff dies.**
- Drop: articles (a/an/the), filler (just/really/basically), pleasantries, hedging.
- Pattern: `[thing] [action] [reason]. [next step]`.
- NOT: "Sure! I'd be happy to help you with that." YES: "Bug in auth middleware. Fix:".
- **Applies to ALL prose:** narration + artifact bodies (spec/ADR/prompt/doc) + **code comments** (comments are prose ‚Üí caveman too).
- **Stays literal (NEVER caveman):** structural data (JSON/YAML keys+values, schemas), identifiers (`R*`/`AC*`/`C*`/`ADR-*`/ids), code syntax. Caveman shortens prose, never corrupts data or breaks code.

## SHARED CONTRACT C ‚Äî re-lock procedure (reused by parts that own a lock: M11.1 skeleton, M11.2 adr, M11.4 aprd)

Editing a frozen body desyncs its lock. Honor immutability = re-seal as a NEW version (a change is a new sealed version, never a silent overwrite):
1. Edit body to caveman + sweep dead paths.
2. Recompute `content_sha256` in Bash with the EXACT algorithm:
   - `aprd  = sha256( aprd.frozen.md )`  ‚Äî single file.
   - `adr   = sha256( concat(sorted(.adr/log/*.md) by name) )`  ‚Äî **drafts NOT hashed**, index/lock NOT hashed.
   - `skel  = sha256( concat(sorted(.hld/skeleton/*) by name) + skeleton.frozen.md )`.
   - (sha256 over the byte-concatenation of the listed files, sorted by filename.)
3. In the `.lock`: set new `content_sha256`, bump `version` `v2`‚Üí`v3`, set `signed_at` to the run timestamp, keep `signer: self-host:agentic-delivery-pipeline`, `status: frozen`.
4. Verify recompute == lock (re-run the sha, compare).
