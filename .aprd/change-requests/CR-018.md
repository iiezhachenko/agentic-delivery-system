## Ask

DIAGNOSE bundles Phase-4 build-red adjudication (Parts A + B) with Phase-0 bugfix intake (Part C: BUGFIX-LOCALIZE). Phase boundary violation (LC2): Part C reads change-requests + aprd.frozen.md, reproduces/localizes/root-causes a defect, writes `.aprd/diagnosis.json` — all Phase-0 intake operations inside a Phase-4 prompt. Block F2 (LC1, 2 unrelated output paths), F3 (LC2, phase boundary crossing), warn F7 (LC4, different primary outputs per mode).
