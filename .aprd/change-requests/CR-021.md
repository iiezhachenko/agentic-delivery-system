# CR-021 — Bugfix-mode-family policy: SYNTHESIZE / IMPLEMENT / MATERIALIZE-ORACLE / VERIFY-OUTPUT

> Parent: CR-016. Operator-directed (2026-06-12). Decision overridden at gate: EXTRACT all. ADR-0037 (D37) revised. Caveman register.

## Ask

Cross-cutting LC4 warn across 4 roles: bugfix mode behaves differently from greenfield/feature-add in each. Design question: extract separate bugfix-class role family OR accept as designed mode-switching. Severity: warn (not block). ADR-0037 (D37) decides.

| role | finding id | bugfix behavior |
|---|---|---|
| SYNTHESIZE.md | F10 | Greenfield: draft + CRITIQUE gate, no aprd.lock write. Feature-add/bugfix: frozen version + re-sign aprd.lock. Different primary outputs. |
| IMPLEMENT.md | F11 | Bugfix: minimal in-place edit of existing BLAST_RADIUS symbol. Greenfield/feature-add: write new component. Opposite behaviors. |
| MATERIALIZE-ORACLE.md | F12 | Bugfix: reproduction + regression layers, supersedes existing slice oracle. Greenfield/feature-add: full contract/flow/acceptance oracle. Different layer sets. |
| VERIFY-OUTPUT.md | F13 | Bugfix: reproduction red→green + scoped regression bar. Greenfield/feature-add: full contract/flow/acceptance/class-ext ladder. Different verification bar. |

## ADR decision

ADR-0037 (D37) revised at operator gate — EXTRACT all bugfix-mode-family roles. Rationale: strict SRP (one-role-one-prompt) + failure isolation (CLAUDE.md "role separation load-bearing") > DRY-skeleton minimization. F10/F11/F12/F13 + F9 Part C → resolved-by-split.

## Scope delta

Extracts 5 new roles. Modifies 5 existing roles. Updates spine routing.

### A — Modify SYNTHESIZE.md

`prompts/00-aprd/SYNTHESIZE.md` — remove feature-add/bugfix path (frozen version + aprd.lock re-sign). Retain greenfield draft path + CRITIQUE gate. Primary output: draft aPRD artifact. Stop condition: CRITIQUE-GATE passes.

### B — New role: SYNTHESIZE-INCREMENT

`prompts/00-aprd/SYNTHESIZE-INCREMENT.md` — new role. Handles feature-add + bugfix delivery classes. Reads existing aprd.frozen.md + change request. Produces frozen version increment + re-signs aprd.lock. No CRITIQUE gate (increment is concrete, not speculative). Dispatch signal: class=feature-add OR class=bugfix, phase=aprd. Note: lumps feature-add + bugfix into single increment role — both share frozen-version + lock-re-sign path; greenfield-vs-increment split is structural boundary, not greenfield-vs-bugfix.

### C — Modify IMPLEMENT.md

`prompts/04-build/IMPLEMENT.md` — remove bugfix path (minimal in-place edit at BLAST_RADIUS symbol). Retain greenfield/feature-add write-new-component path.

### D — New role: IMPLEMENT-BUGFIX

`prompts/04-build/IMPLEMENT-BUGFIX.md` — new role. Bugfix delivery class only. Reads BLAST_RADIUS symbol from `.hld/skeleton/blast-radius.json` (or equivalent). Performs minimal in-place edit targeting BLAST_RADIUS boundary. No new component scaffolding. Dispatch signal: class=bugfix, phase=build.

### E — Modify MATERIALIZE-ORACLE.md

`prompts/04-build/MATERIALIZE-ORACLE.md` — remove bugfix path (reproduction + regression layers). Retain greenfield/feature-add full oracle (contract/flow/acceptance layers).

### F — New role: MATERIALIZE-ORACLE-BUGFIX

`prompts/04-build/MATERIALIZE-ORACLE-BUGFIX.md` — new role. Bugfix delivery class only. Produces reproduction layer (failing test that reproduces defect) + regression layer (scoped regression guard). Supersedes existing slice oracle at BLAST_RADIUS boundary. Dispatch signal: class=bugfix, phase=build.

### G — Modify VERIFY-OUTPUT.md

`prompts/04-build/VERIFY-OUTPUT.md` — remove bugfix path (repro red→green + scoped regression bar). Retain greenfield/feature-add full verification ladder (contract/flow/acceptance/class-ext).

### H — New role: VERIFY-OUTPUT-BUGFIX

`prompts/04-build/VERIFY-OUTPUT-BUGFIX.md` — new role. Bugfix delivery class only. Verifies: (1) repro test goes red→green, (2) scoped regression bar passes. No full class-ext ladder. Dispatch signal: class=bugfix, phase=build.

### I — New role: DERIVE-TESTS-BUGFIX

`prompts/03-hld/DERIVE-TESTS-BUGFIX.md` — new role. Bugfix delivery class only. Mints reproduction test spec + regression layer spec. Source finding: F9 Part C. Note: CR-020 extracts DERIVE-BUILD-DAG from DERIVE-TESTS skeleton pass; CR-021 extracts DERIVE-TESTS-BUGFIX from DERIVE-TESTS bugfix pass. Post-both-CRs, DERIVE-TESTS = skeleton/increment test-specs only (no build-dag, no bugfix pass). CR-020's note that Part C "stays" is superseded by this CR.

### J — Spine updates

`tools/io/io-manifest.json` — add entries for all 5 new roles. Update SYNTHESIZE to remove feature-add/bugfix output path. Update IMPLEMENT to remove bugfix path. Update MATERIALIZE-ORACLE to remove bugfix path. Update VERIFY-OUTPUT to remove bugfix path.

`tools/io/components.json` — add entries for all 5 new roles.

`prompts/_playbooks/bugfix-spine.md` (or equivalent bugfix playbook) — update role routing: chain SYNTHESIZE-INCREMENT (not SYNTHESIZE), IMPLEMENT-BUGFIX (not IMPLEMENT), MATERIALIZE-ORACLE-BUGFIX (not MATERIALIZE-ORACLE), VERIFY-OUTPUT-BUGFIX (not VERIFY-OUTPUT), DERIVE-TESTS-BUGFIX (not DERIVE-TESTS Part C). If no dedicated bugfix playbook exists, note as new artifact to create.
