# CR-025 — Two-tier model fleet: per-hole tier router + local 8B endpoint

> Standalone. Operator-directed (2026-06-13). Workstream: `feature/mcp-adp-powers`. Source: `_self-host/00-analysis.md` §4, `_self-host/01-8b-offload.md` §1–§4. Caveman register.

## Ask

After CR-023+CR-024, per-step model context = ~6–8k (flat on either tier). Two model tiers available:
- **Local self-hosted 8B** — MCP talks to it directly. Zero Claude tokens. Free retries.
- **Claude (Haiku-4-5 default, Sonnet escalation)** — irreducible judgment holes only.

Decisive criterion: **offload to 8B IFF output is gated by deterministic oracle** — schema-valid + value-parity + tests-green + det-lint — so Claude never reads it to trust it. A failed 8B attempt costs only local compute; on stall, server discards + Claude authors fresh (clean-room preserved). Offloading judgment holes is net-negative (Claude reads it anyway → double cost).

Economics: ~70% calls offloadable × ~80% 8B pass-rate ≈ ~55% Claude-token cut, zero quality loss (every output still oracle-gated). Source: `01-8b-offload.md` §1.

## Scope delta

### A — `adp_route_tier` tool

New deterministic MCP tool: `adp_route_tier(role, holePointer) → "local-8b" | "claude"`. Backed by a code-table (not LLM call). Lookup order: (1) per-hole override, (2) per-role class default, (3) global fallback = `claude`. Deterministic, disk-derived, selftestable.

Tier table (from `01-8b-offload.md` §3):

**Class A → default `local-8b`, escalate-on-stall K=3:**
`IMPLEMENT`, `IMPLEMENT-BUGFIX`, `MATERIALIZE-ORACLE`, `MATERIALIZE-ORACLE-BUGFIX`, `INTEGRATE`, `DEMO-GEN`, `EXTRACT`, `EXTRACT-RULES`, `DECISION-EXTRACT`, `SLICE-EXTRACT`, `QUESTION-GEN`, `AUDIT-REPORT`, `AUDIT-PROMOTE`, `RECONCILE (00)`, `VERIFY (00)`, `TRIAGE`, `RE-RANK`, `SEQUENCE*`

**Per-hole split (all roles):** `narration / restatement / verbatim-carry` holes → `local-8b`. `decision / finding / design / adversarial` holes → `claude`. MCP knows hole pointer + hint → applies per-hole rule on top of per-role default.

**Class B → `claude` (no 8B):**
`GAP-DETECT`, `CRITIQUE`, `RECONCILE-CRITIQUE`, `OPTION-GEN`, `EVALUATE-DECIDE`, `DERIVE-COMPONENTS`, `DEFINE-CONTRACTS`, `MODEL-DATA`, `MODEL-FLOWS`, `MAP-NFR`, `SYNTHESIZE`, `SYNTHESIZE-INCREMENT`, `SYNTHESIZE-ADR`, `FOUNDATION-CUT`, `SKELETON-IDENTIFY`, `VERTICALITY-CHECK`, `RESOLVE-LOCAL`, `DIAGNOSE`, `BUGFIX-LOCALIZE`

**Already no-model (Tier-1 emitters, DR-023 `adp_emit`):**
`BASELINE-MAP`, `BUILD-PLAN`, `DERIVE-TESTS`, `VERIFY-OUTPUT`, `DERIVE-BUILD-DAG`

### B — Local 8B endpoint wiring

`adp-server/` gains a configurable endpoint: `LOCAL_8B_URL` (env var). When `adp_route_tier` returns `local-8b`, driver sends packet to local endpoint. When absent/unreachable → fallback to Claude (safe degradation). No spine fork — packet shape identical for both tiers (model = param, not embedded in packet).

### C — Escalation rule (clean-room preservation)

8B stall (K=3 attempts, oracle fails all): server DISCARDS the 8B output entirely. Claude receives the original packet (same as initial dispatch). Clean-room preserved — Claude never sees a bad 8B draft. Optional later: pass failure signature only (small), let Claude author fresh with that hint.

### D — Tier-table selftest

`tools/det/route-tier.selftest.mjs`: for each class-A role + known narration hole → assert `local-8b`. For each class-B role + decision hole → assert `claude`. Planted swap (class-A role → claim `claude`) → FAIL. Both-directions.

## Affected downstream

- Driver (CR-023): calls `adp_route_tier` per hole before dispatching; routes to local or Claude endpoint accordingly.
- `adp_next` (CR-023): step packet unchanged; tier selection is post-packet (driver decides who fills holes).
- `_fixtures/` oracle: unchanged — oracle gates the OUTPUT regardless of tier. Same bar, different model.
- Rollout: R1 (wire endpoint + tier table) → R2 (narration holes only, lowest risk) → R3 (IMPLEMENT + MATERIALIZE on `_fixtures/`, measure pass-rate) → R4 (ratchet per measured pass-rate). Source: `01-8b-offload.md` §6.
